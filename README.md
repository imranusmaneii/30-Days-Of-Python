
# 🐍 30 Days of Python Challenge

Welcome to my **30 Days of Python Challenge**!

This repository documents my journey of learning Python over 30 days. Every day, I'll upload new content including notes, exercises, code examples, and mini-projects. The goal is to build a strong foundation in Python while creating a resource that others can learn from as well.

## 🎯 Objectives

* Learn Python consistently for 30 days
* Strengthen problem-solving skills
* Build coding habits through daily practice
* Create a public learning resource for beginners
* Track progress and growth over time

## 📂 Repository Structure

```text
30-days-of-python/
│
├── Day-01/
├── Day-02/
├── Day-03/
├── ...
└── Day-30/
```

Each day's folder may contain:

* 📘 Notes
* 💻 Practice Code
* 📝 Exercises
* 🚀 Mini Projects
* 📚 Additional Resources
